# Initial ETF codec evidence

## Scope

Owned, bounded encode/decode for the M1 ETF subset: small/integer values, UTF-8 atoms, tuples, binaries, nil, proper lists including `STRING_EXT`, and `NEW_PID_EXT`.

## Safety properties

- version byte required;
- trailing bytes rejected;
- all reads bounds-checked;
- collection, binary, atom, and nesting limits applied before allocation;
- decoded variable-size values have explicit recursive deinitialization;
- improper list tails rejected.

## Verification

```sh
zig fmt --check build.zig src tests examples fixtures/etf/manifest.zig
zig build test-unit
zig build test-conformance
zig build test-all
```

OTP 28 golden vectors decode and re-encode byte-for-byte for the committed stable subset. OTP 25–27 black-box verification remains pending.
